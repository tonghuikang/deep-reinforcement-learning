Probability Mass Function (PMF)

• PMF is a function that gives the probability that a discrete random variable is exactly equal to some value.

Example

• Discrete random variable: $X \in \{1, 3, 7\}$.
• PDF:
    $p(1) = 0.2$,
    $p(3) = 0.5$,
    $p(7) = 0.3$.
